# NexusAI - Personal AI Assistant for Android

Your personal AI assistant with multi-model support, offline capabilities, and elegant design.

## Features

- 💬 Multi-model AI chat (GPT-4, Claude, Gemini)
- 💻 Code assistant (8+ languages)
- 📄 Document analysis
- 🖼️ Image generation
- 🔄 Offline support
- 🎨 4 beautiful themes
- ⚙️ Customizable settings

## Getting Started

1. Clone this repository
2. Install dependencies: `npm install`
3. Start: `npm start`
4. Scan QR code with Expo Go

## Build APK

```bash
eas build --platform android --type apk
